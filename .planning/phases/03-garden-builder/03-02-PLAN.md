<!-- From Plan 01 outputs — the API the frontend consumes -->

Garden API Endpoints (all require auth cookie):
- GET /api/gardens -> Garden[] (user's gardens)
- POST /api/gardens { name, rows, cols } -> Garden (201)
- GET /api/gardens/:id -> { id, name, rows, cols, createdAt, cells: GardenCell[] }
- PUT /api/gardens/:id/cells/:row/:col { plantId } -> { message } (200)
- DELETE /api/gardens/:id/cells/:row/:col -> { message } (200)
- DELETE /api/gardens/:id -> 204

Plant Relationship Endpoints (public, no auth):
- GET /api/plants/companions -> Array<{ plantId: number, companionId: number }>
- GET /api/plants/antagonists -> Array<{ plantId: number, antagonistId: number }>

Existing Plant Endpoint (public):
- GET /api/plants -> Plant[] (all plants with id, name, category, growthForm)

Type shapes:
- Garden: { id: number, name: string, rows: number, cols: number, createdAt: string }
- GardenCell: { id: number, gardenId: number, row: number, col: number, plantId: number, plantName: string }

    - import axios from "axios"
    - export const getGardens = () => axios.get("/api/gardens", { withCredentials: true })

    - export const getGarden = (id) => axios.get(`/api/gardens/${id}`, { withCredentials: true })
    - export const upsertCell = (gardenId, row, col, plantId) => axios.put(`/api/gardens/${gardenId}/cells/${row}/${col}`, { plantId }, { withCredentials: true })
    - export const clearCell = (gardenId, row, col) => axios.delete(`/api/gardens/${gardenId}/cells/${row}/${col}`, { withCredentials: true })
    - export const deleteGarden = (id) => axios.delete(`/api/gardens/${id}`, { withCredentials: true })
    - export const getAllCompanions = () => axios.get("/api/plants/companions")
    - export const getAllAntagonists = () => axios.get("/api/plants/antagonists")

    Create ui/src/pages/Garden.jsx:
    - Auth check on mount: call getMe() from authService, navigate to /login on 401
    - State: gardens (array), selectedGarden (object with cells), plants (array), companions (array), antagonists (array), loading (bool), error (string), showCreateDialog (bool), newGardenName/newRows/newCols, pickerOpen (bool), pickerCell ({row, col, cell} or null)
    - On mount: fetch gardens via getGardens(), fetch plants via axios.get("/api/plants"), fetch companions via getAllCompanions(), fetch antagonists via getAllAntagonists(). Use Promise.all for parallel fetch.
    - Layout: two-panel using MUI Box with display: "flex"

If garden selected, show Typography h5 with garden.name, then render <GardenGrid garden={selectedGarden} cells={selectedGarden.cells} companions={companions} antagonists={antagonists} onCellClick={handleCellClick} />
    - handleSelectGarden(gardenId): call getGarden(gardenId), set selectedGarden to response data (includes cells)
    - handleCreateGarden: validate name not empty, rows/cols 1-20. Call createGarden service. Refresh garden list. Select the new garden.
    - handleCellClick(row, col, cell): set pickerCell to {row, col, cell}, set pickerOpen to true
    - handlePlantSelect(plant): if plant is null, call clearCell. Otherwise call upsertCell with plant.id. Then refresh selected garden data via getGarden(selectedGarden.id). Close picker.
    - Create dialog: MUI Dialog with TextField for name, two TextField type="number" for rows and cols (inputProps min=1 max=20), Button "Create"

    Create ui/src/components/GardenGrid.jsx:
    - Props: garden, cells, companions, antagonists, onCellClick
    - Convert cells array to cellGrid object keyed by "row,col" string
    - Build companionSet: new Set from companions array, each entry keyed as `${Math.min(plantId,companionId)}-${Math.max(plantId,companionId)}`
    - Build antagonistSet: same pattern from antagonists array using plantId/antagonistId
    - Helper pairKey(a, b): `${Math.min(a,b)}-${Math.max(a,b)}`
    - Helper getCellStatus(row, col): if no plant at (row,col) return "empty". Check 4 orthogonal neighbors within bounds. For each occupied neighbor, compute pairKey. If any pair in antagonistSet return "antagonist" (antagonist takes priority). If any pair in companionSet return "companion". Otherwise "neutral".
    - Render Box with sx={{ display: 'grid', gridTemplateColumns: `repeat(${garden.cols}, 64px)`, gap: 0.5 }}
    - Iterate Array.from({ length: garden.rows }) x Array.from({ length: garden.cols }), render GardenCell for each position
    - Do NOT use MUI Grid component — use CSS grid via Box sx prop
