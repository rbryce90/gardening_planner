    Create server/types/garden.d.ts with two types:
    - `Garden`: id (number), userId (number), name (string), rows (number), cols (number), createdAt (string, optional)
    - `GardenCell`: id (number), gardenId (number), row (number), col (number), plantId (number), plantName (string, optional — for joined queries)

    Create server/databases/gardenDb.ts following the plantDb.ts singleton pattern:
    - Import DatabaseSync from "node:sqlite" and logger from "../utils/logger.ts"
    - Module-level dbInstance variable, lazy init in getDatabase()
    - Open "plants.db" (same file as plantDb)
    - PRAGMA foreign_keys = ON
    - CREATE TABLE IF NOT EXISTS gardens (id INTEGER PRIMARY KEY AUTOINCREMENT, user_id INTEGER NOT NULL, name TEXT NOT NULL, rows INTEGER NOT NULL, cols INTEGER NOT NULL, created_at TEXT DEFAULT (datetime('now')), FOREIGN KEY (user_id) REFERENCES users(id))
    - CREATE TABLE IF NOT EXISTS garden_cells (id INTEGER PRIMARY KEY AUTOINCREMENT, garden_id INTEGER NOT NULL, row INTEGER NOT NULL, col INTEGER NOT NULL, plant_id INTEGER NOT NULL, FOREIGN KEY (garden_id) REFERENCES gardens(id) ON DELETE CASCADE, FOREIGN KEY (plant_id) REFERENCES plants(id), UNIQUE (garden_id, row, col))
    - Log "Garden database initialized."
    - Export getDatabase function

    Create server/repositories/gardenRepository.ts as class GardenRepository:
    - Import getDatabase from "../databases/gardenDb.ts" and Garden/GardenCell types
    - async getGardens(userId: number): Promise<Garden[]> — SELECT id, name, rows, cols, created_at FROM gardens WHERE user_id = ? ORDER BY created_at DESC. Map snake_case to camelCase.
    - async createGarden(userId: number, name: string, rows: number, cols: number): Promise<Garden> — INSERT INTO gardens (user_id, name, rows, cols) VALUES (?, ?, ?, ?). Return the created garden by querying with db.prepare("SELECT last_insert_rowid() as id").get() then fetching the row.
    - async getGardenById(gardenId: number, userId: number): Promise<Garden | null> — SELECT with WHERE id = ? AND user_id = ?. Return null if not found.
    - async getGardenCells(gardenId: number): Promise<GardenCell[]> — SELECT gc.id, gc.garden_id, gc.row, gc.col, gc.plant_id, p.name as plant_name FROM garden_cells gc JOIN plants p ON gc.plant_id = p.id WHERE gc.garden_id = ? ORDER BY gc.row, gc.col. Map to camelCase.
    - async upsertCell(gardenId: number, row: number, col: number, plantId: number): Promise<void> — INSERT OR REPLACE INTO garden_cells (garden_id, row, col, plant_id) VALUES (?, ?, ?, ?)
    - async clearCell(gardenId: number, row: number, col: number): Promise<boolean> — DELETE FROM garden_cells WHERE garden_id = ? AND row = ? AND col = ?. Return changes > 0.
    - async deleteGarden(gardenId: number, userId: number): Promise<boolean> — DELETE FROM gardens WHERE id = ? AND user_id = ?. Return changes > 0. ON DELETE CASCADE handles cells.
    - Export singleton: export const gardenRepository = new GardenRepository()

    Create server/controllers/gardenController.ts:
    - Import gardenRepository from "../repositories/gardenRepository.ts"
    - Export async functions that delegate to repository: getGardens(userId), createGarden(userId, name, rows, cols), getGardenById(gardenId, userId), getGardenCells(gardenId), upsertCell(gardenId, row, col, plantId), clearCell(gardenId, row, col), deleteGarden(gardenId, userId)
    - Follow the thin-wrapper pattern from plantController.ts

    Create server/routes/gardenRoutes.ts:
    - Import Router from "express", authMiddleware from "../middleware/auth.ts", controller functions
    - const gardenRouter = Router()
    - gardenRouter.use(authMiddleware) — all routes require auth
    - GET / — req.user!.userId -> getGardens -> 200 with array
    - POST / — validate name (string, required), rows (number, 1-20), cols (number, 1-20) from req.body. Return 400 with { message } on invalid input. Call createGarden -> 201 with garden object.
    - GET /:id — parse id, call getGardenById with userId. If null -> 404. Otherwise call getGardenCells(gardenId), return 200 with { ...garden, cells }.
    - PUT /:id/cells/:row/:col — parse params, validate plantId from body. Call getGardenById for ownership check -> 404 if not found. Validate row < garden.rows and col < garden.cols -> 400 if out of bounds. Call upsertCell -> 200.
    - DELETE /:id/cells/:row/:col — parse params, call getGardenById for ownership -> 404. Call clearCell -> 200 if deleted, 404 if cell was empty.
    - DELETE /:id — parse id, call deleteGarden with userId -> 204 if deleted, 404 if not found.
    - export default gardenRouter

    Modify server/index.ts:
    - Add import: import gardenRouter from "./routes/gardenRoutes.ts"
    - Add app.use("/api/gardens", gardenRouter) BEFORE the app.use(errorHandler) line
